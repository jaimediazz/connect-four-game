RULES

OJECTIVE

Be the first player to connect 4 of the same coloured discs in a row (other vertically, horizontally or diagonally).

HOW TO PLAY

  1. Red goes first in the first game.
  2. Players must alternate turns and only one disc can be dropped each turn.
  3. The game ends when there is a 4-in-a-row or a stalemate.
  4. The starter of the previous game goes second on the next game.
